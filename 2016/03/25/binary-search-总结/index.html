binary search 小结

发表于 2016-03-25   |   分类于 leetcode   |  

binary search是最简单，最入门的算法了，实现也比较简单，变化也可以有很多，所以经常会在面试的时候碰到。之前看过有一个模板，初始化是left=-1, right=arr.length，然后while里的条件是left<right-1，据说能统一处理防止某些情况下越界或是死循环的情况，但我始终不大习惯于这么写，还是觉得自己的方法，也是相当常见的方法，比较好理解。

比如在一个sorted array里找target, 假设array里没有重复元素，一般就会这么写：

public int search(int[] arr, int target) {
    if (arr == null || arr.length == 0) return -1;
    int left = 0, right = arr.length-1;
    while (left <= right) {
        int mid = left + (right - left)/2;
        if (arr[mid] == target) {
            return mid;
        }
        else if (arr[mid] > target) {
            right = mid - 1;
        }
        else left = mid + 1
    }
    return -1;
}

其实理解好了这个最简单的例子，就可以举一反三了。

  • 首先就是要看准可以排除的部分，才能确定是如何更新right或是left。比如上面的例子，如果arr[mid]>target, 意味着target肯定在mid的左边，那mid和mid的右边都可以排除，则right=mid-1。如果是array里有重复的值，要找到第一个等于target的位置，当arr[mid]==target时，我们不能立刻舍弃mid位置的值，因为它有可能就是第一个等于target的位置，所以这时right=mid
  • 当更新left或right时，有right=mid这种时，如果是while(left<=right)就会有死循环，必须改成while(left<right)。而如果有left=mid这种，因为整数相除的关系，即使是while(left<right)还是会死循环，就要改变计算mid的方法，即用mid=left+(right-left+1)/2，这样当left=right-1，即两者想邻时，计算得到的mid就是right的位置，进行left=mid也不会死循环。
  • 有时候还要稍稍注意最后的返回结果的一些corner case，比如所有数字都比target大之类的情况。

leetcode上相关的题有这些：

  1. First Bad Version, 如果mid上是bad version，则mid的右边都可以排除，但mid的位置不可以立刻排除，所以是right=mid

    public int firstBadVersion(int n) {
    int left = 1, right = n;
    while (left<right) {
        int mid = left + (right - left)/2;
        if (isBadVersion(mid)) { //排除mid的右侧
            right = mid;
        }
        else {
            left = mid+1; //mid肯定不是要找的位置，排除mid及其左侧
        }
    }
    return left;
}

  2. Search Insert Position, 相当于返回第一个大于等于target的位置。当A[mid]<target时，可以放心地排除mid及mid的左边，所以left=mid+1。最后的返回值是left, 因为left指向的永远是大于等于target的值，也可以用只有一个element的数组进行简单验证

    public int searchInsert(int[] A, int target) {
    int left = 0, right = A.length-1;
    while (left <= right) {
        int mid = left + (right - left)/2;
        if (A[mid] == target) {
            return mid;
        }
        else if (A[mid]<target) {
            left = mid + 1;
        }
        else {
            right = mid - 1;
        }
    }
    return left;
}

  3. Search for Range, 这里就涉及到寻找右边界，需要改变mid计算方式。此时，如果nums[mid]<=target，意味着mid的左边可以排除，但mid位置还不能立刻排除，所以就需要left=mid。

    public int[] searchRange(int[] nums, int target) {
    if (nums == null || nums.length == 0) return new int[]{};
    int left = 0, right = nums.length-1;
    while (left < right) {
        int mid = left + (right-left)/2;
        if (nums[mid] >= target) {
            right = mid;
        }
        else {
            left = mid + 1;
        }
    }
    if (nums[left] != target) return new int[]{-1,-1};
    int res1 = left;  //得到左边界
    left = 0;
    right = nums.length-1;
    while (left < right) {
        int mid = left + (right - left + 1)/2;
        if (nums[mid]<=target) {
            left = mid;
        }
        else {
            right = mid - 1;
        }
    }
    return new int[]{res1, left};
}

  4. Find Peak Element，用提到的两种方法分别计算mid1, mid2, 注意更新left, right时要用哪一个mid的值

    public int findPeakElement(int[] nums){
    if (nums == null || nums.length == 0) return 0;
    int left = 0, right = nums.length-1;
    while (left <= right) {
        int mid1 = left + (right - left)/2;
        int mid2 = mid1 + 1;
        if (nums[mid1] < nums[mid2]) {
            left = mid2; //peak的位置肯定在mid2或它的右侧
        }
        else {
            right = mid1;
        }
    }
    return left;
}

  5. Find Minimum in Rotated Sorted Array, 关键是要拿mid跟right的值进行比较，如果nums[mid]<nums[right]，mid（不包括mid）到right的部分就可以排除，否则，排除另一半。

    public int findMin(int[] nums) {
    if (nums == null || nums.length == 0) return -1;
    int left = 0, right = nums.length - 1;
    while (left < right) {
        int mid = left + (right - left)/2;
        if (nums[mid] < nums[right]) {
            right = mid;
        }
        else {
            left mid + 1;
        }
    }
    return nums[left];
}

  6. Find Minimum in Rotated Sorted Array II，跟上面的方法一样，就是在nums[mid]==nums[right]的时候，right–，因为只能排除right位置上的值。

    public int findMin(int[] nums) {
    if (nums == null || nums.length == 0) return -1;
    int left = 0, right = nums.length - 1;
    while (left < right) {
        int mid = left + (right - left)/2;
        if (nums[mid] < nums[right]) {
            right = mid;
        }
        else if (nums[mid] > nums[right]) {
            left = mid + 1;
        }
        else {
            right -- ;
        }
    }
    return nums[left];
}

  7. Search in Rotated Sorted Array，通过比较A[left]和A[mid]，分成两种情况，如果A[left]<=A[mid]，（当left和right相邻时，计算得到的mid就是left的位置，所以等于必须放在这里），说明left到mid之间是单调递增的，如果发现target在它们之间，就可以排除mid和它右边的部分，否则的话，就可以排除mid和它左边的部分。另一种情况就是A[left]>A[mid]时，那mid和right之间就是单调递增的，同样进行处理。这一题的关键还是找到单调递增的区域。

    public int search(int[] A, int target) {
    if (A == null || A.length == 0) return -1;
    int left = 0, right = A.length - 1;
    while (left <= right) {
        int mid = left + (right - left)/2;
        if (A[mid]==target) return mid;
        if (A[left] <= A[mid]) {
            if (A[left] <= target && target < A[mid]) {
                right = mid - 1;
            }
            else {
                left = mid + 1;
            }
        }
        else {
            if (A[mid] < target && target <= A[right]) {
                left  = mid + 1;
            }
            else {
                right = mid - 1;
            }
        }
    }
    return -1; 
}

  8. Search in Rotated Sorted Array II, 跟上面一样，就是在A[mid]==A[left]的时候，left++，因为只能排除当前的left不包含想要的结果。

    public boolean search(int[] nums, int target) {
    if (nums == null || nums.length == 0) return false;
    int left = 0, right = nums.length - 1;
    while (left <= right) {
        int mid = left + (right - left)/2;
        if (nums[mid] == target) return true;
        if (nums[mid] > nums[left]) {
            if (nums[left] <= target && target < nums[left]) {
                right = mid - 1;
            }
            else {
                left  = mid + 1;
            }
        }
        else if(nums[mid] < nums[left]) {
            if (nums[mid] < target && target <= nums[right]) {
                left = mid + 1;
            }
            else {
                right = mid - 1;
            }
        }
        else {
            left ++ ;
        }
    }
    return false;
}

  9. Sqrt(x), 也可以用类似binary search的方法

    public int sqrt(int x) {
    if (x == 0) return 0;
    int left = 1, right = x/2 + 1;
    while (left <= right) {
        int mid = left + (right - left)/2;
        if (mid == x/mid) return mid;
        else if (mid > x/mid) { //防止溢出
            right = mid - 1;
        }
        else {
            left = mid + 1;
        }
    }
    return right; //right指向较小的值，由于返回整数取floor
}
